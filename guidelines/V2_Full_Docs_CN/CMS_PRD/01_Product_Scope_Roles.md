# 1. 产品范围与系统定位

## 1.1 产品定位
此内容管理系统（CMS）是专为机上娱乐（IFE）内容生命周期设计的核心运营平台。主要实现了基于松下 (Panasonic) MMA 规范的内容元数据 (Metadata) 管理、航线与设备配置 (Profile Setup) 以及可视化的内容编排。

## 1.2 目标受众与适用范围
- **目标用户**：平台旨在服务于航司内容运营团队、发布与运维团队以及第三方内容提供商（CSP）。
- **系统边界**：此CMS处理地面端内容管理与配置映射，不涉及机上播放器 UI 的直接渲染。

## 1.3 核心产品价值
1. **自主可控与数据主权**：实现机上内容流水线的完全本地化，支持自定义的树形结构分类编辑器 (Tree Builder) 和全局搜索。
2. **精细化编排**：通过 Profile, Route, Class, Category Builder 实现极高自由度的航线与客舱定向投放。

# 2. 访问控制与角色系统
目前系统通过 `AuthContext.tsx` 在前端实现了基础的角色路由隔离。

## 2.1 角色模型
1. **内容编辑/提供商 (IFE_PROVIDER / CSP)**：被授予添加、编辑和预览内容记录的权利。数据域被控制在其负责的内容范围内。
2. **航司运营经理 (AIRLINE)**：统筹分类树结构，定义复杂的可见性规则。对业务编排享有全局权限。

## 2.2 租户数据隔离策略
通过不同的登陆角色与 `company_id`，系统前端提供隔离的界面体验。后台也同步限制了越权拉取和篡改非本租户核心数据的能力。
